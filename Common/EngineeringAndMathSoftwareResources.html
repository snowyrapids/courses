Engineering & Math Software Resources

Mathcad Family

Mathcad/Prime is great for working with units and provides a nice "What you see is what you get" (WYSIWYG) interface. A free alternative is SMath Studio.

Mathematica Family

Some free alternatives include the following:

  • wxMaxima
  • Jupyter Notebooks is actually an editor with a Mathematica-like interface. You can use several programming languages with it; Python is a very common one.

Matlab Family

Some free alternatives include the following:

  • FreeMat
  • GNU Octave

Maple Family

Technically, Maxima inspired Maple. Take a look at wxMaxima under the "Mathematica family" module.

HTML5

Do calculations in your web browser; no extra installation necessary! 

HTML5, (sometimes called Dynamic HTML), refers to using three main technologies together -- HTML, CSS, & JavaScript -- to make a web page.
