Assignments

Students are primarily responsible for the software/library installations necessary to complete the assignments. Help might be found through the Resources link, professor, lab assistants, and online search engines (a.k.a, "Google it!"). However, the student's assignment code must be his/her own; no copying allowed! Copying code — including markup, comments, etc. — for an assignment is generally considered to be cheating, and the corresponding assignment will be flunked.
