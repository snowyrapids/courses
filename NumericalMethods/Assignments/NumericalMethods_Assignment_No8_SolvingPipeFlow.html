Numerical Methods Assignment #8

Engineering Application: Solving Simultaneous Equations in Pipe Flow Calculations

This assignment will be done only in HTML5 (HTML, CSS, & JavaScript). You will use your Ridders' Method toolbox and your Gauss-Seidel code from your previous assignments to solve the required equations behind an HTML user interface.

The equations and description of the pipe-flow problem are found in study_guide_U2_day3.pdf on the Math & Numerical Methods Resources —> Other Professors' Resources page. Look for the boxed equations; you will need to solve them simultaneously using the Gauss-Seidel method, just as you did for the non-linear simultaneous equations of your previous assignment. Also notice that the friction factor $f$ must be solved for using a root finder; you will use Ridders' Method for this step.

The task includes the following requirements:

  • Use a radio button that allows the user to determine which variable is being solved for, the volumetric flow-rate $\dot{V}$ or the diameter $D$.
  • All variable values must be taken from user-input boxes (e.g., $P_1$ & $P_2$, not just those being solved for). In addition to the variables in the equations, you must also allow the user to specify the initial guesses for $\dot{V}$ and $D$, the bounds for the Ridders' Method function, the separate maximum iterations for the Gauss-Seidel and Ridders' Method solvers, and the relaxation factors for all equation variables being updated in the Gauss-Seidel loop (not just $\dot{V}$ and $D$).
  • As there will be many input boxes, you must organize the user-interface layout in a logical manner to make it user-friendly. Use descriptions as necessary.
  • Your solver code must be contained in a separate & adjacent (in the same folder) JavaScript file. Your other script functions can also be stored in the JS file as desired. Do not forget to place both the HTML and the JS files together in the same zip file.
  • Your web page must include default values in all of the input boxes. Use the values provided by the homework problem located at the end of the aforementioned PDF file. For the relaxation factors, set the defaults to the values that give you the lowest number of iterations in your solvers (i.e., that make your code converge the fastest).
  • The solver code must be started when the user clicks a solve button button (as has been done in the previous assignments).
  • The results must be clearly displayed on the web page along with the number of iterations performed by the Gauss-Seidel process and the maximum number of iterations used by the Ridders' Method (out of all the times it was called; hint: simply store the maximum count in a variable until needed).

Also be certain that in your assignment you...

  • Submit your assignment as a zip file.
  • Correctly use style (see the style guide link on the Language Resources page), adequate commenting (see the commenting link on the Language Resources page; don't forget class and function comment blocks), and well-named variables.
  • Remove unnecessary or unused code.
  • Name your submission file according to the naming convention on the Numerical Methods Assignments page.
