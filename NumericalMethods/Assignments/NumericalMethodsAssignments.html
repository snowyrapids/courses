Naming Your Files

Include your name in the names of submitted files. For instance, rename your file to be "LastName_FirstName_assignment2.ipynb". The order of the last and first name will help in grading. You can rename a Jupyter notebook file while it is open by simply clicking on the file name at the top.

Using Jupyter Notebooks

You can use Jupyter Notebooks when developing your JavaScript code by starting each JavaScript cell with %%javascript. To display output from your JavaScript code, use element.append() in a similar manner to the way you use print() in your Python code cells.
