Caution: Carefully read and observe all of the assignment instructions so that you do not lose points for missed requirements.

Ridders' Method (Root-finding) Toolbox

This assignment requires that you use HTML5 (JavaScript, HTML, & CSS) to program Ridders' Method. Before submitting the assignment, remember to verify that your program meets the following criteria:

  • Use a class (a "prototype" in JavaScript) & member methods/data to house your solver's code. Also use an object to hold the current equation that you are solving; pass the equation object as a parameter to your solver method.
  • Use "setter" and "getter" methods when you access member data from outside the root-finder object. Include "set" and "get" in the names of such methods. Your root-finding method, however, can return the result to the calling statement outside the object. In other words, do not access member variables directly; access them through member methods (class member functions) instead. However, try to avoid having to access member data from outside the object in the first place!
  • Use Ridders' method from your prototype to accurately find roots for the assigned equation(s) and display the results.
  • Make your root-finding method as generic as possible so that you can use it with different equations without having to create a new solver object.
  • Allow the initial bracket values of Ridders' method to be set with the method parameters (in the function call).
  • Correct use of style (see the style guide link on the Language Resources page), adequate commenting (see the commenting link on the Language Resources page; don't forget class and function comment blocks), and well-named variables.
  • No unnecessary or unused code.
  • Your code must work correctly.
  • Name your submission file according to the naming convention on the Numerical Methods Assignments page.

Using Ridders' Method

You will submit an HTML document (with the ".html" extension) for this assignment. Create a separate section of the page for each of the following tasks; use headings, horizontal rulers, etc. as needed.

Task #1

Your name must be displayed at the top of the page.

Task #2

Use your root-finder object to solve each of the following equations. Each equation should be in its own section. Each equation section must have its own "Solve" button. Remember that the user might want to solve an equation again after changing the input parameters; this should be permitted without having to reload the page.

The default accuracy must be to at least 4 decimal places. User input/parameters must be entered through input boxes; the default values must be displayed. The equation being solved must be included and rendered in its corresponding section using LaTeX and MathJax. Your web page must work in both Chrome/Chromium and Firefox. Remember that you will be submitting only one file.

Task #2.1

Find the positive zero of the polynomial $$x^5-2 x^2 -4.$$ User parameters (entered through input boxes) are the root-finder brackets, the absolute error, and the maximum iterations allowed. Remember to display the default values for these as well.

Task #2.2

Find the second positive root of $cos(x)$. The user requirements are the same as the previous task.

Task #2.3

Find the friction factor $f$ in the Colebrook-White equation $$\frac{1}{\sqrt{f}}=-2 log\left(\frac{\varepsilon}{3.7D_h}+\frac{2.51}{Re\sqrt{f}}\right).$$ In addition to including the user parameters of the previous sections, also provide input boxes for the following parameters with their associated default values:

  • $\varepsilon$ = 1.5e-4
  • $D_h$ = 0.315
  • $Re$ = 6000
