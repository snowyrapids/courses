Numerical Methods Assignment #3

Monte Carlo Toolbox

This assignment must use Python 3 and the Jupyter notebook template provided below. Before submitting the assignment, verify that your program meets the following criteria:

  • Use a class & member methods/data to house your Monte Carlo integration code. (Data must be on the lowest scope practical; in other words, do not make variables like a function-level counter effectively "global" to the class by declaring it a class member variable).
  • Use constructor methods to initialize the object upon instantiation.
  • Use "setter" and "getter" methods to interface with member data from outside the object. Include "set" and "get" in the names of such methods. Your integration method, however, can return the integral result to the calling statement outside the object.
  • Use a Monte Carlo method from your class to accurately integrate the assigned equation(s) and print the result. This means that the method (i.e., a member function to a class) must use a random number generator; otherwise it wouldn't be a Monte Carlo method (meaning a numerical method). (The "random" library that should have been included with your installation would be a good place to look for one.) To obtain an accurate integration result, you will need a lot of random points; do not store them altogether in a variable (that could use a lot of memory).
  • Make your Monte Carlo method as generic as possible so that you can use it with different equations.
  • Allow the number of seed/Monte Carlo points to be set with the method parameters (in the function call).
  • Use matplotlib to graph the original equation (noted below) and 100 Monte Carlo points together. This does not have to use a class method like the one you must use to accurately integrate the equation, as you might have to store the points in order to plot them. However, it should also print the (likely inaccurate) result so that it can be compared to the (hopefully) more accurate class method that you will have coded.
  • Correct use of style (see the style guide link on the Language Resources page), adequate commenting (see the commenting link on the Language Resources page), and well-named variables.
  • No unnecessary or unused code.
  • Your code must work correctly.

The accuracy must be to at least one decimal place. The assigned integrals and equations to integrate are as follows:

  1. $$\int_0^{\pi/2} cos(x)\,dx$$
  2. $$\int_0^{10} e^{-1/x}\,dx$$
  3. With an interval of $\left[-\pi/2,10\right]$, $$f(x)=\begin{cases} cos(x) & x\lt 0 \\ e^{-1/x} & 0\leq x \end{cases}$$
Only use equation 2 for the plotting requirement; you do not need to plot the other equations.
