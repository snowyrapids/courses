Numerical Methods Assignment #7

Solving Simultaneous Equations

This assignment will have two parts. You will do each part in both HTML5 and Python 3. Use the provided Jupyter Notebook template for your Python code and an HTML file for your HTML5 code. You will, therefore, be submitting both files to I-Learn wrapped in one zip file. The Jupyter notebook template for the Python portion of this assignment can be downloaded here.

Part 1: Solving Simultaneous Equations with $A x = B$

The system of equations you will solve for this part of the assignment is $$\begin{cases} x + y - z = 7 \\ 4x - y + 5z = 4 \\ 2x + 2y - 3z = 0 \end{cases}$$

Using HTML5

Use a Math.js function to invert and multiply the arrays/matrices. (Note that Math.js is different from the built-in Math library in JavaScript, even though they have similar names.) Source the minified version at "https://cdnjs.cloudflare.com/ajax/libs/mathjs/3.9.1/math.min.js" in your HTML head section (between the "<head></head>" tags.

Display the original system of equations using MathJax and LaTeX. Trigger your JavaScript code using a button, and display the results.

Using Python 3

Display the equations being solved using a markdown cell. Use NumPy arrays and functions to invert and multiply your matrices. Also display the results.

Part 2: Solving Non-Linear Simultaneous Equations With a Gauss-Seidel Method

Solve the following non-linear system of equations to an accuracy of 4 decimal places using an iterative Gauss-Seidel method wrapped in a loop: $$\begin{cases} x = \sqrt{\frac{120-y}{8}} \\ y = x^3-1 \end{cases}$$ Look in the Other Professors' Resources on the class website for more help.

Using HTML5

Display the original equations using MathJax and LaTeX. The user should be able to specify the initial guesses, maximum number of iterations, and absolute error using input boxes; the input boxes should supply default values that will solve the equations. Trigger your JavaScript code using a button, and display the results.

Using Python 3

Display the equations being solved using a markdown cell. Specify the initial guesses, maximum number of iterations, and the absolute error using variables at the beginning of your code cell. Display the results.
