Numerical Methods Assignment #1

Installing Python3, requisite libraries, and Jupyter

To receive credit for this assignment, you must demonstrate to your professor in person (and obtain acknowledgment of it being recorded) that your computer can correctly run the following files:

  • ME342_testscript.py – The python test script
  • ME342_testNotebook.ipynb – The Jupyter Notebook test file
  • Note: You might have to right-click on the link and select "Save link as...".

The software and libraries to run these files must be installed on your personally-owned or managed computer. This could be either your laptop, your remotely-accessed desktop, or a virtual machine running on such in which you have installed the requisite software/libraries yourself. Online web services, such as Jupyter's "Try it in your browser" link, are not acceptable.

To run the python file (ending in ".py"), the following software and libraries must be installed:

  • Python 3.x (version 3.4 or newer)
  • NumPy library
  • matplotlib plotting library
  • SymPy symbolics library
Jupyter is an editor that is displayed in your web browser while running an interpreter on your computer. To run the Jupyter Notebook file (ending in ".ipynb"), the following must additionally be installed:
  • jupyter

You could also install Python 3 and the libraries as a package, such as Anaconda (huge) or Miniconda (not quite as huge). This method might be easier for many users. More help and links are provided on the Software Resources page.
