Numerical Methods Assignment #2

Simulated Coin Tosses

For this assignment you will program a simulated coin toss in Python 3 using Jupyter. The number of coin tosses should be assigned to a variable at the beginning, and the results should be displayed at the end. Such results include the numbers and percentages of heads and tails. To obtain full credit the following must be included in your notebook:

  • Adequate commenting (see the commenting link on the Language Resources page)
  • Correct use of style (see the style guide link on the Language Resources page)
  • The coin result must be determined by a random number generator (check out the "random" library that should have been included with your installation)
  • Your simulation must include a function that you created, a branching statement, and well-named variables
  • The results must be printed as output to the cell, e.g., print("Number of heads:", headCount). Be certain to include the percent sign % on the printed percentages.
  • No unnecessary or unused code
  • Your simulator must work correctly

The Jupyter notebook template for this assignment can be downloaded here. You will submit your completed notebook file to the appropriate drop-box in I-Learn.
