Numerical Methods Assignment #10

Solving Multiple ODE's with the Runge-Kutta Method

This assignment will be done only in Python 3 with a Jupyter-Notebook. You will use the template provided here.

You will create an ODE toolbox with a class; it must contain a member function of the 4th order Runge-Kutta Method that is capable of solving two 1st order differential equations simultaneously. The differential equations' functions must be encapsulated together (two equation methods in one object) within an equation object that will be passed as a parameter to the Runge-Kutta method along with any other pertinent parameters. You must preserve code encapsulation by instantiating the ODE toolbox only once for the entire assignment and by avoiding outside alteration of the toolbox's member data (either directly or through getter/setter functions).

After solving the ODE's with the 4th-Order Runge-Kutta method, you must plot their respective solutions against the independent variable using matplotlib. Your plot must additionally contain a legend and appropriate axes titles.

The assigned ODE's that your code must correctly solve simultaneously are as follows:

  • $\frac{dx}{dt}=-0.5 x + y t$
  • $\frac{dy}{dx}=4-0.1 x -0.2 y$
  • $x(0)=1$
  • $y(0)=2$
  • $t$ ranges from 0 to 2
  • The number of steps is 15
Print the results of $x(t_{final})$ and $y(t_{final})$ in the appropriate code cell after running your solver. Be certain to check your results with either your hand calculations or another software package.
