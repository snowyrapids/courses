Secant Method (Root-finding) Toolbox

This assignment must use Python 3 and the Jupyter notebook template provided below. Before submitting the assignment, verify that your program meets the following criteria:

  • Include a PDF file of your Top-Down Design diagram for your Secant Method program with your Jupyter Notebook file in your assignment submission. The diagram must be clearly written and use a "bubble" format, although you do not need to draw circles around your wording.
  • Use a class & member methods/data to house your solver's code. (Data must be on the lowest scope practical; in other words, do not make variables like a function-level counter effectively "global" to the class by declaring it a class member variable).
  • Use constructor methods to initialize the object upon instantiation (if needed).
  • Use "setter" and "getter" methods when you access member data from outside the object. Include "set" and "get" in the names of such methods. Your root-finding method, however, can return the result to the calling statement outside the object. In other words, do not access member variables directly; access them through member methods (class member functions) instead.
  • Use a secant method from your class to accurately find roots for the assigned equation(s) and print the results.
  • Make your secant method as generic as possible so that you can use it with different equations without having to create a new solver object.
  • Allow the initial guesses of the secant method to be set with the method parameters (in the function call).
  • Correct use of style (see the style guide link on the Language Resources page), adequate commenting (see the commenting link on the Language Resources page), and well-named variables.
  • No unnecessary or unused code.
  • Your code must work correctly.

Using the Secant Method

You might consider plotting the equations with matplotlib to see where you should set the initial values for the Secant Method. The result's accuracy must be to at least six decimal places. The assigned equations and associated roots to solve for are as follows:

  1. Find the positive zero of the polynomial $$x^5-2 x^2 -4$$
  2. Find the second positive root of $cos(x)$
  3. Find the friction factor $f$ in the Colebrook-White equation $$\frac{1}{\sqrt{f}}=-2 log\left(\frac{\varepsilon}{3.7D_h}+\frac{2.51}{Re\sqrt{f}}\right),$$ with the following values:
    • $\varepsilon$ = 1.5e-4
    • $D_h$ = 0.315
    • $Re$ = 6000
